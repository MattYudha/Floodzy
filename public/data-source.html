Data Intelligence Sources

Floodzy functions as a specialized data aggregator, synthesizing inputs from government satellites, ground sensors, and verified open APIs to generate realtime warnings.

01 // BMKG Open Data

Primary source for weather forecasts, rainfall intensity predictions, and extreme weather warnings across the Indonesian archipelago.

REFRESH: HOURLY / 3-HOURLY

02 // Ground Telemetry (IoT)

Direct inputs from Floodzy's pilot network of ultrasonic water level sensors installed in key river basins in Jabodetabek.

REFRESH: REALTIME (< 5s LATENCY)

03 // Satellite Altimetry

Processing data from Sentinel-3 and Jason-3 for broader river discharge estimation and topological mapping support.

REFRESH: DAILY PASS

04 // WAQI (Air Quality)

Supplementary environmental data provided by the World Air Quality Index project for comprehensive situational awareness.

REFRESH: HOURLY

> AGGREGATOR WARNING
